# run-ejs
EJSの仕様を確認するために使います。
|コマンド|内容|
|-|-|
|`npm i`|インストール|
|`npm start`|起動|

## 基本的な使い方
https://ejs.co/

## console
コンソールに出力させることができます。変数の確認などに便利です。  
`table`や`count`も利用できます。
```html
<% console.log('hoge'); %>
<!-- hogeとターミナルに出力される -->
```

## EJSを利用する際便利なVSCode拡張機能
- .ejs
